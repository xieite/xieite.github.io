xieite::rotate_swap()

Defined in fragment xieite:rotate_swap
(header-only: <xieite/algo/rotate_swap.hpp>)

Swaps the values of multiple references by rotating their values to the left.
When rotating by multiple spaces, internally uses the least possible number of temporary copies.


Declaration

 
template<std::size_t n = 1, typename... Args>
constexpr std::tuple<Args&...> rotate_swap(Args&... args)
noexcept((... && (xieite::is_nothrow_move_ctor<Args> && xieite::is_nothrow_move_assign<Args>)));


Example

 
import std;
import xieite;

int main() {
	int a = 1;
	int b = 2;
	int c = 3;
	xieite::dump(a, b, c);

	xieite::rotate_swap(a, b, c);

	xieite::dump(a, b, c);
}
Output: 
1 2 3
2 3 1
