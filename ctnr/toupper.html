xieite::toupper()

Defined in header <xieite/ctnr/toupper.hpp>

Converts every character of a string (or a single character) to uppercase.
Only handles the English alphabet.


Declarations

 #0 
template<xieite::is_ch Ch = char>
[[nodiscard]] constexpr Ch toupper(Ch c) noexcept;
Makes a single character uppercase. Internally uses a lookup table.

#1 
template<typename Ch = char, typename Traits = std::char_traits<Ch>, typename Alloc = std::allocator<Ch>>
[[nodiscard]] constexpr std::basic_string<Ch, Traits, Alloc> toupper(std::basic_string_view<Ch, Traits> str) noexcept;
Makes an entire string uppercase. Internally calls #0.


Example

 
int main() {
	std::string_view text = "HeLlO, wOrLd!";

	xieite::dump(xieite::toupper(text));
}
Output: 
HELLO, WORLD!
