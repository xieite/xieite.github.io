xieite::strlen()

Defined in header <xieite/ctnr/strlen.hpp>

A constexpr equivalent of std::strlen().


Declaration

 #0 
template<typename Str>
requires(std::is_pointer_v<Str> && xieite::is_ch<std::remove_pointer_t<Str>>)
[[nodiscard]] constexpr std::size_t strlen(Str str) noexcept;
Uses a requires clause to make this overload less preferred by the compiler. Invokes undefined behavior if the passed string is not null-terminated.

#1 
template<xieite::is_ch Ch, std::size_t size>
[[nodiscard]] constexpr std::size_t strlen(Ch(&)[size]) noexcept;
Returns size - 1. The compiler should prefer this overload for string literals.


Example

 
int main() {
	xieite::dump(xieite::strlen("abcdef"));
}
Output: 
6
