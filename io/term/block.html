xieite::term::block()

Defined in header <xieite/io/term.hpp>

Enables or disables input blocking (waiting for user input before reading the input buffer).


Declaration

 
void block(bool x) noexcept;


Example

 
int main() {
	xieite::term term;
	term.block(false);
	std::puts(term.read_str().c_str());
}
