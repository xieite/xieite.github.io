xieite::term::clear_screen_from()

Defined in header <xieite/io/term.hpp>

Clears everything on the screen starting from the cursor's position.


Declaration

 
void clear_screen_from() noexcept;


Example

 
int main() {
	xieite::term term;
	std::puts("Hello,");
	std::puts("world!");
	term.mv_cursor(-1, 0);
	term.clear_screen_from();
}
Possible output: 
Hello,
