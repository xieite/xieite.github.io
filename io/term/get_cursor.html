xieite::term::get_cursor()

Defined in header <xieite/io/term.hpp>

Gets the cursor's position.


Declaration

 
[[nodiscard]] xieite::pos get_cursor() noexcept;


Example

 
int main() {
	xieite::term term;
	auto [row, col] = term.get_cursor();
	std::println("({}, {})", row, col);
}
Possible output: 
(0, 0)
