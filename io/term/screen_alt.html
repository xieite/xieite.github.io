xieite::term::screen_alt()

Defined in header <xieite/io/term.hpp>

Enables or disables the alternative screen buffer.


Declaration

 
void screen_alt(bool x) noexcept;


Example

 
int main() {
	xieite::term term;
	term.screen_alt(true);
	std::print("Hello, ");
	term.screen_alt(false);
	std::puts("world!");
}
Possible output: 
       world!
