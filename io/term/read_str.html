xieite::term::read_str()

Defined in header <xieite/io/term.hpp>

Reads the entire input stream buffer to a std::string.


Declaration

 
[[nodiscard]] std::string read_str() noexcept;


Example

 
int main() {
	xieite::term term;
	std::string str = term.read_str();
	std::puts(str.c_str());
}
Possible output: 
hello
