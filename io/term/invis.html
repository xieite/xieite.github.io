xieite::term::invis()

Defined in header <xieite/io/term.hpp>

Enables or disables invisible text.


Declaration

 
void invis(bool x) noexcept;


Example

 
int main() {
	xieite::term term;
	std::puts("Hello,");
	term.invis(true);
	std::puts("world!");
}

Hello,
