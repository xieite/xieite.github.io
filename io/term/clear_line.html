xieite::term::clear_line()

Defined in header <xieite/io/term.hpp>

Clears the line on which the cursor is currently.


Declaration

 
void clear_line() noexcept;


Example

 
int main() {
	xieite::term term;
	std::fputs("Hello, ", stdout);
	term.clear_line();
	std::puts("world!");
}
Possible output: 
       world!
