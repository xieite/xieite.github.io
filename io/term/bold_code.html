xieite::term::bold_code()

Defined in header <xieite/io/term.hpp>

Returns a std::string which enables or disables bold text when printed.


Declaration

 
[[nodiscard]] static constexpr std::string bold_code(bool x) noexcept;


Example

 
int main() {
	std::println("{}Hello, world!", xieite::term::bold_code(true));
}
Possible output: 
Hello, world!
