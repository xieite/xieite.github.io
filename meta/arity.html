xieite::arity<>

Defined in header <xieite/meta/arity.hpp>

Finds the arity of a tuple-like or aggregate type.


Declaration

 
template<typename T>

Example

 
struct S {
	int a;
	int x[2];

	int y[5];

static_assert(xieite::arity<S> == 5);
