xieite::end<>

Defined in fragment xieite:end
(header-only: <xieite/meta/end.hpp>)

A utility for marking the end of a template parameter list to allow defining type aliases after.


Definition

 
template<typename>
concept end = false;


Example

 
import std;
import xieite;

template<typename A, typename B, xieite::end..., typename C = std::common_type_t<A, B>>
void foo(A, B, C, C) {}

int main() {
	foo(1, 2.3, 4.5, 6.7);
}
