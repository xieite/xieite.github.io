xieite::any::any()

Defined in fragment xieite:any
(header-only: <xieite/meta/any.hpp>)

Constructs a xieite::any.


Declaration

 #0

Defaulted default-constructor for the compiler to do implicit magic.

#1 
constexpr any(auto&&...) const noexcept;
Allows constructing from anything, for convenience.


Example

 
import xieite;

int main() {
	xieite::dump(requires { xieite::any(); });
	xieite::dump(requires { xieite::any(418); });
	xieite::dump(requires { xieite::any('h', nullptr, [] {}); });
}
