xieite::type_list<Ts...>::append_list<>

Defined in fragment xieite:type_list
(header-only: <xieite/meta/type_list.hpp>)

Appends the contents of some type container to the back of a xieite::type_list.


Declaration

 
template<typename List>
using append_list = /* ... */;
Uses append<> internally.


Example

 
import xieite;

int main() {
	using foo = xieite::type_list<int, char>;
	using bar = xieite::type_list<float, double>;

	xieite::dump(xieite::name<foo::append_list<bar>>());
}
Possible output: 
xieite::type_list<int, char, float, double>
