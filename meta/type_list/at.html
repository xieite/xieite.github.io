xieite::type_list<Ts...>::at<>

Defined in header <xieite/meta/type_list.hpp>

Selects an element of Ts by its index.


Declaration

 
template<std::size_t idx>
using at = /* Ts...[idx] */;


Example

 
int main() {
	using List = xieite::type_list<int, char, float, double>;

	xieite::dump(xieite::type_name<List::at<2>>);
}
Possible output: 
float
