xte::is_int<>

Defined in header <xte/trait/is_int.hpp>

Checks that a given type is a signed or unsigned integer type, excluding bool.


Definition

template<typename T>
concept is_int = xte::is_unsigned<T> || xte::is_signed<T>;



Example

static_assert(xte::is_int<int>);
static_assert(xte::is_int<unsigned int>);
static_assert(not xte::is_int<bool>);
