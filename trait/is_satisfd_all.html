xieite::is_satisfd_all<>

Defined in fragment xieite:is_satisfd_all
(header-only: <xieite/trait/is_satisfd_all.hpp>)

Specifies that a functor is satisfied by all of several types.


Definition

 
template<auto fn, typename... Ts>
concept is_satisfd_all = (... && xieite::is_satisfd<fn, Ts>);


Example

 
import std;
import xieite;

int main() {
	static constexpr auto fn = []<std::integral> {};

	xieite::dump(xieite::is_satisfd_all<fn, int, char, short>);
	xieite::dump(xieite::is_satisfd_all<fn, void*, double, long>);
}
Output: 
true
false
