xieite::add_rref<>

Defined in header <xieite/trait/add_rref.hpp>

A shorter alias for std::add_rvalue_reference_t<>, named consistently with other utilities in this library.


Declaration

 
template<typename T>
using add_rref = std::add_rvalue_reference_t<T>;
