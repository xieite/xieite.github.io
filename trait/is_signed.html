xte::is_signed<>

Defined in header <xte/trait/is_signed.hpp>

Checks that a given type is a signed integer type.


Definition

template<typename T>
concept is_signed = std::signed_integral<T>;



Example

static_assert(xte::is_signed<int>);
static_assert(not xte::is_signed<unsigned int>);
