XIEITE_IF()

Defined in header <xieite/pp/if.hpp>

Called in three parts (condition, then, else). If the first argument expands to 1, expands to the second argument, otherwise expands to the third argument.


Definition

 #if 
#define XIEITE_IF(_x) XIEITE_IIF(XIEITE_BOOL(_x))

#iif 
#define XIEITE_IIF(_b) /* ??? */


Example

 
XIEITE_IF(1)('a')('b')
XIEITE_IF(0)('a')('b')
Preprocessor output: 
'a'
'b'
