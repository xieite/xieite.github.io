XIEITE_LIFT(), XIEITE_LIFT_LOCAL(), XIEITE_LIFT_ONE(), XIEITE_LIFT_ONE_LOCAL(), XIEITE_LIFT_MEMBER(), XIEITE_LIFT_MEMBER_PTR()

 Defined in header <xieite/lift.hpp>
(header-only: <xieite/pp/lift.hpp>)

Wraps a function in a lambda object.


Definitions

 #lift 
#define XIEITE_LIFT(...) /* []??? { ??? } */
Expands to a lambda expression with no capture-default and variadic parameters.

#lift-local 
#define XIEITE_LIFT_LOCAL(...) /* [&]??? { ??? } */
Expands to a lambda expression with a reference capture-default and variadic parameters.

#lift-one 
#define XIEITE_LIFT_ONE(...) /* []??? { ??? } */
Expands to a lambda expression with no capture-default and one parameter.

#lift-one-local 
#define XIEITE_LIFT_ONE_LOCAL(...) /* [&]??? { ??? } */
Expands to a lambda expression with a reference capture-default and one parameter.

#lift-member 
#define XIEITE_LIFT_MEMBER(...) /* []??? { ??? } */
Expands to a lambda expression which calls a member of its first parameter with the rest of its parameters.

#lift-member-ptr 
#define XIEITE_LIFT_MEMBER_PTR(...) /* []??? { ??? } */
Same as #lift-member but accesses a member through operator->().


Example

 
#include <xieite/fwd.hpp>
#include <xieite/lift.hpp>
import std;

void run(auto&& fn) {
	std::invoke(XIEITE_FWD(fn), "Hello, world!\n");
}

int main() {
	run(XIEITE_LIFT(std::printf));
}
Output: 
Hello, world!
