xieite::pow_of_2()

Defined in header <xieite/math/pow_of_2.hpp>

Checks if an integer is a power of 2. The algorithm is stolen from [Bit Twiddling Hacks].
This feature is deprecated; This page will no longer be updated. Use std::has_single_bit() instead.


Declaration

 
template<std::unsigned_integral T>
[[nodiscard]] constexpr bool pow_of_2(T value) noexcept;


Example

 
int main() {
	xieite::dump(xieite::pow_of_2(4u));
	xieite::dump(xieite::pow_of_2(0u));
}
Output: 
true
false
