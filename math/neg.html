xieite::neg()

Defined in header <xieite/math/neg.hpp>

Checks whether a number is negative. Always returns false for unsigned integers.
Can be used to avoid warnings enabled by -Wtype-limits.


Declaration

 
template<xieite::is_arith T>
[[nodiscard]] constexpr bool neg(T value) noexcept;


Example

 
int main() {
	xieite::dump(xieite::neg(5u));
}
Output: 
false
