xieite::unroll()

Defined in header <xieite/fn/unroll.hpp>

Unrolls a constant-sized loop using a std::integer_sequence {}.
Passes the indices to a given functor as a non-type template parameter pack.

Example

 
int main() {
	xieite::unroll<5>([]<int... i> {
		xieite::dump(i...);
	});
}
