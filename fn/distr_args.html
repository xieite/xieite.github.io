xieite::distr_args()

Defined in fragment xieite:distr_args
(header-only: <xieite/fn/distr_args.hpp>)

Uniformly distributes arguments across multiple calls to a functor.


Declaration

 
template<std::size_t arity, typename F, typename... Args>
constexpr void distr_args(F&& fn, Args&&... args) noexcept(false);
Assumes that fn is invocable with arity number of args because checking this is a pain.
Likewise, does not specify noexcept.
Internally checks that arguments are evenly distributable across functor calls with static_assert().


Example

 
import std;
import xieite;

int main() {
	xieite::distr_args<2>(XIEITE_LIFT(xieite::dump), 1, 2, 3, 4, 5, 6);
}
Output: 
1 2
3 4
5 6
