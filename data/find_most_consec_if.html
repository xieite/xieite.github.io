xieite::find_most_consec_if()

Defined in header <xieite/data/find_most_consec_if.hpp>

Returns a view to the longest consecutive subrange for which a predicate functor returns true.

Example

 
int main() {
	int array[] = { 0, 0, 2, 1, 0, 0, 0, 4, 0 };

	xieite::dump(xieite::find_most_consec_if(array));
}
