xte::ptr_diff()

Defined in header <xte/data/ptr_diff.hpp>

Computes the absolute difference between two pointers, avoiding signed integer wrapping.


Definition

[[nodiscard]] constexpr xte::uz ptr_diff(auto* first, decltype(first) last, xte::uz max) noexcept;



Example

constexpr xte::fixed_array data = { 1, 2, 3 };
static_assert(xte::ptr_diff(data.begin(), data.end(), data.size()) == 3);
